# Xmy-Free.github.io
星梦游Xmy原创项目作品


项目在线预览非常卡，
最好下载在本地电脑文件中运行，
最为推荐使用火狐浏览器打开，
第一次打开加载页面需要些许时间，
劳烦耐心等待。



页面中几乎所有的交互事件通过鼠标产生

单击/长按/拖动  -  
鼠标左键 /鼠标中键/鼠标右键/滑动鼠标滚轮，
会有意想不到的效果。

特别说明：
长按鼠标中键可返回流浪地球三体主页，
ctrl＋鼠标滚轮可以缩放页面大小
